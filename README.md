# PokerBot
This is a Poker bot for Discord


## Code-Style
* All curly braces should be on it's own line
```js
if (true)
{
	while (true)
	{
		
	}
}
```
* Instead of this
```js
if (true) {

	while (true) {
	
	}
}
```

## Usage
* Install node.js if not already installed
* cd into the folder
* npm install discord.js@12.5.3 
* Execute node bot.js
